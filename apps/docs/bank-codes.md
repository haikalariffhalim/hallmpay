## Notes

1. **Islamic bank arms** generally share the same IBG code and routing as their parent conventional bank. They are listed separately for DuitNow BIC purposes.
2. **FPX IDs** — Retail mode (01) is for B2C consumer transactions. Corporate mode (02) is for B2B transactions.
3. **Digital banks** were licensed by BNM in 2022 and have been progressively launching since 2023-2024.
4. **ICBC special note**: Actual account number is 19 digits with prefix "01". For IBG transfers, omit the "01" prefix and use the remaining 17 digits.
5. **DuitNow Acquirer IDs** are from PayNet's official acquirer list. Not all banks have separate acquirer IDs — some only participate via their BIC code.
6. **E-wallets** — "Pay-to-Proxy" means payment via DuitNow ID (phone number, etc.). "Pay-to-Account" means direct transfer to the wallet's account number.
