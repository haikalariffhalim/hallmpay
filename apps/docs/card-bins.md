> Bank Identification Numbers for Malaysian-issued Visa and Mastercard cards.

## Overview

- **142 BINs** across **27 card issuers**

- **116 credit** + **26 debit** cards
- Country: `MY` — Currency: `MYR` (Malaysian Ringgit)

## What is a BIN?

The first 6 digits of any credit or debit card number are the **BIN (Bank Identification Number)**, also called **IIN (Issuer Identification Number)**. It identifies:
- The **card network** (Visa starts with 4, Mastercard with 5)
- The **issuing bank**
- The **card type** (credit / debit / prepaid)

## MyDebit Note

Malaysia's domestic debit scheme **MyDebit** (operated by PayNet) is co-branded on Visa/Mastercard networks. MyDebit cards share the same BINs listed here — there is no separate MyDebit-only BIN range. Over 46 million MyDebit cards are in circulation.

## Usage in Code

```javascript
const bins = require('./data/card-bins.json');

// Find issuer by card number
function getIssuer(cardNumber) {
  const prefix = cardNumber.replace(/\s/g, '').substring(0, 6);
  return bins.bins.find(b => b.bin === prefix) || null;
}

// Example
const card = getIssuer('5196 0312 3456 7890');
// → { bin: "519603", network: "mastercard", type: "debit", issuerNormalized: "CIMB Bank" }
```
