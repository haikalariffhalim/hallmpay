## How to Use in Our Repo

Fork SnorSnor9998/Payment-Icon with attribution, then:

1. Copy the SVG files into our `icons/` directory
2. Rename to match our bank IDs (e.g., `Banks/Cimb/Cimb_V1_SQU.svg` → `icons/banks/cimb-bank.svg`)
3. Pick one canonical version per bank (V1 is usually the latest logo)
4. Create a mapping file `icons/icon-map.json` that links bank IDs to icon paths
5. Create missing icons for HIGH priority items
